# PSYCHOBYL Press Kit — Kullanım Kılavuzu

Bu klasör 3 farklı format içerir, hangisini istersen onu kullanabilirsin:

## 📁 İçindekiler

```
PRESS_KIT/
├── README.md                  ← Bu dosya (sen okuyorsun)
├── index.html                 ← STANDALONE WEB PRESS KIT (önerilen)
├── press_kit_EN.md            ← Notion için İngilizce (paste-ready)
├── press_kit_TR.md            ← Notion için Türkçe (paste-ready)
└── assets/
    ├── branding/              ← Logo, capsule, hero art
    └── screenshots/           ← 8 in-game screenshot (1920x1080)
```

## 🚀 Hangisini Kullanmalıyım?

### Seçenek A — HTML Press Kit (en pro) ⭐

**`index.html`** dosyası kendi başına çalışan profesyonel bir web sayfası.

**Avantajlar:**
- ✅ Tek bir URL ver, gazeteci her şeyi görsün
- ✅ EN/TR dil switcher var (butonlarla)
- ✅ Tüm screenshot ve logolar embed edilmiş
- ✅ Atmosferik tasarım (yeşil neon teması)
- ✅ Mobile uyumlu

**Nasıl hostlarsın? (ücretsiz seçenekler):**

#### Option 1: GitHub Pages (en kolay, profesyonel URL)
1. https://github.com/new — yeni public repo aç (örn. `psychobyl-presskit`)
2. Bu `PRESS_KIT` klasöründeki **tüm dosyaları** repo'ya yükle (drag & drop)
3. Repo Settings → Pages → Source: main branch → root
4. 2-3 dakika sonra URL hazır: `https://YOURUSERNAME.github.io/psychobyl-presskit/`
5. Bu URL'i press email'lerinde kullan

#### Option 2: Netlify (drag & drop)
1. https://netlify.com → free hesap aç
2. Sites → drag & drop ile `PRESS_KIT` klasörünü bırak
3. Anında URL gelir: `https://random-name.netlify.app`
4. Settings'ten custom domain ayarlanabilir

#### Option 3: Kendi domain (boishagames.com aldıysan)
1. cPanel/FTP ile dosyaları yükle
2. URL: `https://boishagames.com/psychobyl-presskit/`

---

### Seçenek B — Notion Press Kit (en hızlı)

**`press_kit_EN.md` ve `press_kit_TR.md`** Notion'a yapıştırılabilir markdown dosyaları.

**Adımlar:**
1. https://notion.so → free hesap aç (yoksa)
2. **+ New Page** → ismi "PSYCHOBYL Press Kit (EN)"
3. `press_kit_EN.md` dosyasını metin editörde aç → tüm içeriği kopyala
4. Notion sayfasına yapıştır → Notion otomatik formatı algılar
5. Aynı şeyi TR için tekrarla
6. Sayfa açıkken sağ üst köşe → **Share** → **Publish to web** → URL'i kopyala
7. Bu URL'i press email'lerinde kullan

**Asset'leri Notion'a eklemek için:**
- Screenshots → Drag & drop ile her bölüme yükle
- Logo → header'a embed et

---

### Seçenek C — Sadece Markdown (basit)

Eğer hosting istemiyorsan, gazetecilere `.md` dosyalarını + `assets/` klasörünü direkt **ZIP halinde** gönderebilirsin.

**Adımlar:**
1. `PRESS_KIT` klasörünü ZIP'le (sağ tık → Send to → Compressed folder)
2. Press email'inde "Press kit attached (12 MB)" diye ekle
3. Düşük downloadable kalır, ama gazeteciler genelde tercih etmez (link ister)
